# 🏐 Volley Odds Calculator

Профессиональный калькулятор для расчета коэффициентов на волейбольные матчи.

## 🚀 Как использовать

1. Введите статистику команд:
   - Название команды
   - Выигранные/проигранные сеты
   - Набранные/пропущенные очки (опционально)

2. Выберите домашнюю и гостевую команду

3. Получите прогноз с двумя методами расчета:
   - PR (на основе очков)
   - BT (на основе сетов)

## 📊 Методы расчета

- **PR (Points Ratio)** - более чувствителен к текущей форме
- **BT (Betting)** - консервативный метод на основе сетов
- **Final** - усредненный результат
